# Headphones Landing Page

## Overview
This project is a stylish and responsive landing page for headphones, designed to provide an engaging user experience across different screen sizes. The layout adjusts dynamically for mobile and desktop users, ensuring seamless interaction.

## Features
- **Responsive Design**: The webpage automatically switches to the mobile version when the screen width is **480px or less**.
- **Hover Effects**:
  - Links hover/active color: **#FF6565**.
  - Button hover/active: **opacity: 0.9**.
- **Centered Layout**: Maximum width of the content is **1000px**, ensuring a clean and well-balanced appearance.
- **Typography**: Uses **Source Sans Pro** and **Spin Cycle OT** fonts.
